# Setting up a Drupal host #

## Requirements ##
    Virtualbox => https://www.virtualbox.org/wiki/Downloads
    Vagrant    => http://downloads.vagrantup.com

## Setup ##
    vagrant up
    vagrant provision

## Using Drupal ##
    your drupal site will be available via http://localhost:8080.
